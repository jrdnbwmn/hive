# Hive

This is a compilation of Claude Code tools that I've built and use for my projects.

# Principles

- **Pareto principle.** Simplify. Don’t over-engineer your setup or workflow. Claude Code can accomplish a large percentage of what you want by itself with no additional tools (and it’s only going to keep getting better). Simplicity means speed, understanding, stability, and future-proofing. However, don’t simplify at the cost of losing out on important features and improvements.
- **Always prioritize context management.** Keep instructions as concise as possible. My context clearing tools are more effective than the vanilla `/compact` — use them often. Mermaid diagrams help a lot with keeping context down.
- **Self improvement.** Build self improvement into the system. Continuously learn and apply lessons from sessions. Make Claude record learnings as it goes along.
- **TDD**
- **YAGNI & DRY**
